1 Let's start with choosing your desired level of security

SECURE

  • running on https
  • automatic certificate generation
  • automatic server name generation
Secure and easy to setup. Uses automatically generated server certificate.
  • running on https
  • manual certificate generation
Best to use with corporate configurations and other requiring the certificate to satisfy specific parameters.

UNSECURE

  • ready to use in no time
  • unsecured, running on http
  • authentication disabled
Perfect for development and enviroments not requiring any level of security.
In order to disable this guide and setup the server manually, please find the 'settings.json' file in your RavenDB installation directory and change the Setup.Mode value to None
