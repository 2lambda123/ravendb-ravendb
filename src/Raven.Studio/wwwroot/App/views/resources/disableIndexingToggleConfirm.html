all indexes on this database.
After clicking Disable, the database must be reloaded in order for this indexing configuration to be in effect.
Without reloading, the change will NOT apply (despite being written to the database record).

There are two ways to reload the database:

  • Disable and then enable the database from the databases view in the Studio.
    This will reload the database on all the cluster nodes immediately.
  • Restart RavenDB on all nodes.
    The database settings configuration will become effective per node that is restarted.
