'Document refresh' will update a document by removing @refresh flag from metadata when given time has passed.
To set the refresh time enter the appropriate date in the metadata @refresh property. The document will be automatically updated at the specified time - @refresh flag will be removed.
As a result:
  • a document could be re-indexed
  • the ETL subscription could be triggered
Note: RavenDB scans which documents are to be updated at a frequency specified.
The actual refresh time can increase (up to) that value.
